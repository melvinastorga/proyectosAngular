Date Pipe

Este pipe funciona para manejar las fechas/horas etc.

  • Fecha Actual
  • Fecha Actual | date
  • Fecha Actual | date: 'short'
  • Fecha Actual | date: 'long'
  • Fecha Actual | date: 'full'
  • Fecha Actual | date: 'MMMM'
  • Fecha Actual | date: 'dd MMMM, yyyy'
  • fecha | date:'long': 'GMT-5'
  • fecha | date:'long': 'GMT-4'
  • fecha | date:'long': '': 'en'
  • fecha | date:'long': '': 'fr'

  • {{fecha | date}}
  • {{fecha | date:'short'}}
  • {{fecha | date:'long'}}
  • {{fecha | date:'full'}}
  • {{fecha | date:'MMMM'}}
  • {{fecha | date:'dd MMMM, yyyy'}}
  • {{fecha | date:'long': 'GMT-5'}}
